# 🦷 med_st — Бот для записи в стоматологию

Два бота в одном проекте:
- **@med_st** — для пациентов (запись на приём)
- **Бот врача** — для просмотра и управления записями

---

## Как работает

### Пациент пишет боту:
1. `/start` — приветствие
2. Нажимает **📅 Записаться на приём**
3. Вводит **ФИО** (обязательно)
4. Вводит **дату рождения** (обязательно)
5. Вводит **телефон** (по желанию, можно пропустить)
6. Выбирает **дату** из тех, что ты добавил
7. Выбирает **время**
8. Выбирает **вид работы**
9. Подтверждает запись
10. Тебе приходит уведомление в Telegram

### Ты управляешь датами:
В **боте врача** командой `/addslot` добавляешь доступные даты.
Пациенты видят только эти даты. Других данных они не видят.

## Настройка

### 1. Создай ДВА бота через @BotFather

**Бот для пациентов:**
- Название: `Стоматология` (или любое)
- Username: `med_st_bot`
- Скопируй токен

**Бот для врача:**
- Название: `Стомат. Врач`
- Username: `med_st_admin_bot`
- Скопируй токен

### 2. Узнай свой Telegram ID
Напиши @userinfobot → скопируй число

### 3. Создай .env файл
```
BOT_TOKEN=токен_бота_для_пациентов
ADMIN_BOT_TOKEN=токен_бота_для_врача
ADMIN_CHAT_ID=твой_telegram_id
GOOGLE_CREDENTIALS=содержимое_serviceAccountKey.json
```

### 4. Запуск на Railway

1. Загрузи на GitHub
2. Подключи к Railway
3. Добавь переменные из .env в Railway Variables
4. Start Command: `python3 main.py`

---

## Команды бота врача

| Команда | Описание |
|---------|----------|
| `/today` | Записи на сегодня |
| `/tomorrow` | Записи на завтра |
| `/schedule` | Все предстоящие записи |
| `/slots` | Доступные даты для записи |
| `/addslot` | Добавить новую дату |
| `/help` | Справка |

---

## Добавление дат (каждый месяц)

В боте врача:
1. Напиши `/addslot`
2. Введи дату: `25.03.2024`
3. Введи времена: `09:00, 10:00, 11:00, 14:00, 15:00, 16:00`
4. Готово — пациенты уже могут записаться!
